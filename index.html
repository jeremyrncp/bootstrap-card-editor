Bootstrap Card Editor

Bootstrap Card Editor is a simple card editor for business intelligence dashboard or other usage.

You have possibility to integrate editor in DOM element or use modal editor

With Bootstrap Card Editor you configure a static card or dynamic card with JSON integration.



After configuration, Bootstrap Card Editor dispatch an event with configuration for render card in any part of your application.
